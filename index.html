Основы математики

Математика - это наука о структурах, порядке и отношениях.

Арифметические операции

  • Сложение (+): a + b = c
  • Вычитание (-): a - b = c
  • Умножение (×): a × b = c
  • Деление (÷): a ÷ b = c

Порядок операций

Правило: скобки → умножение/деление → сложение/вычитание
